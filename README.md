# ギタータブ譜ビューア 🎸

ギターのタブ譜(ASCIIタブ形式)を大きく見やすく表示し、テンポに合わせて
楽譜が自動でスクロールするビューアです。オレンジ色のバーが「いま弾く位置」を示し、
再生中は楽譜のほうが流れていきます。

## 使い方

`index.html` をブラウザで開くだけで動きます。サーバーもインストールも不要です。
スマホ・タブレットでも動作します。

1. 「タブ譜を編集・貼り付け・保存」を開いて、6弦分のASCIIタブ譜を貼り付ける
2. 「読み込む」を押す(複数段のタブ譜は自動で1本につながります)
3. テンポ(BPM)と「1列の長さ」(4分/8分/16分音符)を曲に合わせる
4. ▶ 再生 を押すと、再生位置に合わせて楽譜が自動スクロール

## 機能

- **きれいな譜面描画 (SVG)**: 弦・小節線・フレット番号をSVGで描画。
  h/p/スライドなどの奏法記号も小さく表示され、小節番号も右上に出ます
- **拡大表示**: スライダーで文字サイズを 14〜46px に調整
- **自動スクロール**: BPM と音価設定に基づいて再生位置バーが進む
- **A-B区間リピート**: `A` `B` ボタン(またはキー)で区間を設定すると、
  その区間だけを繰り返し再生。苦手なフレーズの反復練習に
- **位置ジャンプ**: 楽譜をタップ/クリックするとそこから再生
- **メトロノーム**: チェックを入れると拍ごとにクリック音(1拍目はアクセント)
- **ファイル保存/読み込み**: タブ譜をテキストファイルとして保存・再読み込み。
  最後に読み込んだタブ譜はブラウザに自動保存され、次回開いたとき復元されます
- **キーボード操作**: `スペース` 再生/停止、`←` `→` 位置移動、`Home` 先頭へ、
  `A` `B` リピート区間設定、`Esc` リピート解除

## タブ譜の形式

一般的なASCIIタブ譜(1弦=1行、6行1組)に対応しています。

```
e|--------3---3---5---5---3-------|
B|1---1---------------------------|
G|--------------------------------|
D|--------------------------------|
A|--------------------------------|
E|--------------------------------|
```

行頭の弦名(e B G D A E)はあってもなくても読み込めます。
複数の段(システム)は自動的に横一列に連結されます。
